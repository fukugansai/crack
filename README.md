# crack
ひび割れのシミュレーション

ゴムで繋がった正方格子があって、そのゴムがだんだん弱くなってきてある限界の長さになると切れる。切れたところの節は移動してさらに切れやすくなってどんどん裂け目が広がっていくというシミュレーションです。
パラメータを変えるとこんな絵というか動画GIFが出力されます。
